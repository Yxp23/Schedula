# Schedula

A full-stack course planning platform built for Penn State students.

🚧 **In active development**

## What it does

Schedula helps students plan their course schedules smarter with real-time search, filtering, and an ML-powered recommendation engine that suggests courses based on collaborative filtering and NLP sentiment analysis.

## Tech Stack

- **Frontend:** React, Next.js, TypeScript, Tailwind CSS
- **Backend:** FastAPI, SQLAlchemy ORM
- **Database:** PostgreSQL (3 normalized tables)
- **ML:** scikit-learn (collaborative filtering + sentiment analysis)

## Features (in progress)

- Real-time search and filtering across the full course catalog
- Sub-100ms API response times under concurrent load
- Dark/light mode with animated UI
- ML-based course recommendations
- Responsive design for mobile and desktop
